# 🏄‍♂️ Go-Surf — Surfboard Shop Landing Page

**[Посмотреть сайт вживую (GitHub Pages)](https://merlxz.github.io/GO-SURF/)**

Адаптивный лендинг для магазина серфбордов. Проект разработан с нуля(кроме дизайна) для демонстрации навыков семантической верстки, работы с препроцессорами и написания интерфейсной логики. Возможно где-то недоработки, либо что-то сделано не красиво! 



---

## 🚀 Что было сделано

* **Адаптивный слайдер:** Реализована интерактивная карусель товаров, написанная с помощью библиотеки Swiper.js.
* **Архитектура стилей:** Использован SCSS. Весь проект строго структурирован по методологии **БЭМ** (по крайней мере я пытался), что делает код чистым, независимым и легко масштабируемым.
* **Responsive Design:** Полная адаптация верстки под мобильные устройства, планшеты и десктопы.
* **Внимание к деталям:** Реализованы плавные CSS-анимации, переходы и интерактивные ховер-эффекты для кнопок и карточек.

## 🛠 Что использовалось

* **HTML5** (Семантическая разметка)
* **SCSS / CSS3** (Flexbox, CSS Grid, Variables, Swiper)
* **Методология BEM** * **Vanilla JavaScript** (DOM манипуляции, логика слайдера)

## 👨‍💻 Как запустить локально

Если вы хотите запустить проект на своем компьютере:
1. Склонируйте репозиторий: 
   `git clone https://github.com/merlXZ/GO-SURF.git`
2. Откройте папку проекта в вашем редакторе кода.
3. Откройте файл `index.html` в браузере или используйте расширение Live Server в VS Code.
