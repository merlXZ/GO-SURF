# 🏄‍♂️ Go-Surf — Surfboard Shop Landing Page

**[Live Preview(GitHub Pages)](https://merlxz.github.io/GO-SURF/)**

A responsive landing page for a surf shop. The project was built from scratch (excluding the design) to demonstrate skills in semantic markup, working with preprocessors, and writing frontend logic. There might be some imperfections or areas that could look better! I am still working on the adaptive design.

## 🚀 Key Features & Implementation

* **Adaptive Sliders:** All sliders were implemented using the Swiper.js library.
* **Styles Architecture:** Built with SCSS. The entire project is strictly structured using the BEM methodology (or at least I tried), making the code clean, independent, and easily scalable.
* **Responsive Design:** Full layout adaptation for mobile devices, tablets, and desktops.
* **Attention to Detail:** Implemented smooth CSS animations, transitions, and interactive hover effects for buttons and cards.

## 🛠 Tech Stack

* **HTML5** (Semantic Markup)
* **SCSS / CSS3** (Flexbox, CSS Grid, Variables, Swiper)
* **BEM Methodology** * **Vanilla JavaScript** (DOM manipulation, Swiper.js)

## 👨‍💻 How to Run Locally

If you want to run this project on your computer:

1. Clone the repository: 
   `git clone https://github.com/merlXZ/GO-SURF.git`
2. Open the project folder in your code editor.
3. Open the `index.html` file in your browser or use the Live Server extension in VS Code.
